# 🗂️ Kanban Board Project

## 📌 Overview
This is a fully functional Kanban Board web application that allows users to manage tasks across three stages: **To Do**, **Doing**, and **Done**.

The goal of this project was to build a responsive and interactive task management app using **HTML, CSS, and JavaScript**, while working with an external API and implementing local storage for data persistence.

## ✨ Features

- 📥 Fetches tasks from an external API  
- 💾 Saves tasks to localStorage for persistence  
- 🧩 Dynamically renders task columns (To Do, Doing, Done)  
- ➕ Add new tasks  
- ✏️ Edit existing tasks  
- 🗑️ Delete tasks with confirmation  
- 🎯 Priority-based task sorting (High, Medium, Low)  
- 🌙 Light/Dark mode toggle (saved in localStorage)  
- 📱 Fully responsive design with mobile sidebar  

---

## 🧠 Key Concepts Implemented

- **API Integration** using `fetch()`
- **Data Normalisation** for consistent task handling
- **State Management** using localStorage
- **DOM Manipulation** for dynamic UI rendering
- **Event Handling** for interactivity
- **Responsive Design** using CSS media queries
- **Basic Security** using HTML escaping to prevent XSS
